# 숫자 맞추기 게임

1부터 100 사이의 숫자를 맞추는 간단한 웹 게임입니다.

## 사용 방법

1. 숫자를 입력하고 '확인' 버튼을 누르세요.
2. 결과 메시지를 보고 정답을 찾으세요.
3. '다시 시작' 버튼으로 게임을 재시작할 수 있습니다.

## 배포

이 프로젝트는 [GitHub Pages]를 통해 배포됩니다.  
`main` 브랜치의 루트에 있는 파일들이 자동으로 공개됩니다.
